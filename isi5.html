Web Mobile

Mobile Web atau laman web mudah alih adalah serupa dengan mana-mana laman web lain kerana ia berasaskan pelayar laman HTML dan diakses melalui Internet (untuk mudah alih rangkaian biasanya WiFi, 3G atau 4G). Ciri-ciri yang jelas yang membezakan laman web mudah alih dari sebuah laman web standard adalah hakikat bahawa ia direka untuk paparan yang lebih kecil dengan antara muka skrin sentuh.

Seperti mana-mana laman web, laman web mudah alih boleh memaparkan kandungan teks, data, imej dan video. Namun ciri terbaru HTML5 mampu mengakses ciri-ciri mudah alih tertentu seperti klik untuk panggilan (untuk mendail nombor telefon), sms atau pemetaan berasaskan lokasi.

Maqetaa

Maqetta is an open source technology initiative at Dojo Foundation that provides WYSIWYG tooling in the cloud for HTML5 (desktop and mobile). Maqetta allows User Experience Designers (UXD) to perform drag/drop assembly of live UI mockups.

One of Maqetta’s key design goals is to create developer-ready UI mockups that promote efficient hand-off from designers to developers. The user interfaces created by Maqetta are real-life web applications that can be handed off to developers, who can then transform the application incrementally from UI mockup into final shipping application. While we expect the Maqetta-created mockups often will go through major code changes, Maqetta is designed to promote preservation of visual assets, particularly the CSS style sheets, across the development life cycle.
